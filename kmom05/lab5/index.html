Mouse-events

  • Single click on an element to select it. Click again to deselect it.
  • Double click to trigger transition to remove item

Key-events

  • 'E': change element's form (square to circle. circle to square).
  • 'Q': decrese size of selected element(s).
  • 'W': increase size of selected element(s).
  • 'R': change the color of selected element(s).
  • 'T': duplicate selected element(s).
  • 'A': decrease selected element(s)'s z-axis.
  • 'S': increase selected element(s)'s z-axis.
  • 'Y': delete the selected element(s).
  • 'U': deselect all element(s).
  • 'I': select all element(s).
  • 'P': create a random element (form, color, position).
  • 'D': manipulate selected element(s) randomly.
  • 'Arrow(s)': Move the selected element(s) in arrow direction.
